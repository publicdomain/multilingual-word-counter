v0.1.0 notes:
- Hotkeys:
CTRL+F6 = Clipboard word count
CTRL+SHIFT+F6 = Show program window
CTRL+F7 = Native speech time
CTRL+SHIFT+F7 = Foreign speech time
CTRL+F8 = Decrease speed
CTRL+SHIFT+F7 = Increase speed
